\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classEngine_1_1Application}{Application}} abstract class is the core of a C++ application with this engine. 

Inherit this class and call the \mbox{\hyperlink{classEngine_1_1Application_a9b39ff638087f1dbd02a1dace63b2bd0}{start()}} function in your main.
